12 May 2020

Crossedfall updated:

  • Adds documentation to the automated changelog generator script

Hyperio updated:

  • Added holowarrant item to display arrest and search warrants
  • Tweaked Warrant Assistant application for holowarrant use
  • Fixed electric chair not working

MadmanMartian updated:

  • Fixes robot equipment that would use the internal cell, not doing so.
  • Fixes the blitzshell being on accessible camera networks.

Smokingsquirrel updated:

  • new prefix "d" for german language

11 May 2020

Autisem updated:

  • A sink for NeoTheologists hydroponics
  • Both gene machines are added to NoeTheologists hydroponics
  • Atropa belladonna, a new berry which contains atropine
  • Atropine, which stops death and heals oxy damage, can be OD'd and will cause oxy damage when it does
  • fixed roach taming not working, it can now be done with potato's
  • Genes are now unobfuscated, meaning that it will say what they do instead of the guessing game

06 May 2020

Kurgis updated:

  • Changes how gun icons are pathed code-wise, this will have no noticeable effect in-game.
  • Inspector stats adjusted to focus less on Biology skill. MedSpec stats adjusted to focus on Biology skill instead of Vigilance.
  • Gunnery Sergeant given MedSpec access, as it's essentially an extension of the Brig. Inspector given MedSpec access, as it's important for forensics.
  • New folder sprites.
  • AI now properly receives cryosleep respawn bonus when using the 'Wipe Core' command, and entering Intelligence Storage.
  • Added new sprites for existing material stack items, that being steel, plasteel, plastic, iron, wood, and all glass.
  • Removed replaced material sprites from the above addition.
  • Removed unused fcardholder icon.
  • New Ian sprite done by Watler, in addition of a puppy sprite I did based off it.

TorinoFermic updated:

  • Pinpointer will point to nuclear bomb if the disk has been loaded into it.
  • Items will now eject from gibbed roaches.
  • Mechs won't despawn with a occupant while going out of current map.
  • Cleaned up the weird coffin code.
